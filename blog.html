Hello World!

I started this blog for the following reasons:

  • I needed to design a responsive homepage for a project in this web development course.
  • I miss the html and css playground that was MySpace circa 2006-2007.
  • I wanted a place to publish my personal writings, projects, etc.

The first week or so I'll probably be linking and adding some essential content to provide context to the Cursebreakers project, supplimentary studies and surrounding dramas that are worth blogging about.

Among the things I plan to post on this site:

  • Links to resources, tools and references that I have found helpful.
  • Books I am currently reading and/or highly recommend reading.
  • Updates about Cursebreakers and my journey as a founder.
  • Updates in general about my studies, research, projects, etc.
  • Whatever else that I decide the world needs to know about my life.

Currently, there is no plan to use a mailing list or social media outlet to broadcast this blog. Updates will be posted to this site only.

Readers are encouraged to get in touch to discuss any related (or unrelated) topics. Should you have any questions or feedback, feel free to send an email to hello@cursebreakers.net

Until next time,

-Esau
